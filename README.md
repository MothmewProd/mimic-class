# mimic-class
Repository for the PF2e Mimic Class
https://scribe.pf2.tools/v/BlMd1yt2-mimic-v8

Macros folder is useful for helping during online play. 
Run MimicLib at game start/reset. Use mimic-intake to prepare abilities. 
Use mimic-perform to use abilities. 
Use mimic-catalogue to surface current abilities in catalogue and or effect of current stance.
Use clear-cat to reset catalogue between fights.
